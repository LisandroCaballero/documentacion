TRACKING MASIVO

Objetivo:

El objetivo es que el cliente pueda realizar el rastreo / tracking de sus paquetes de forma masiva a través de un webservice y obtenga los distintos estados por cada una de las piezas.

PARÁMETROS DE ENTRADA:

Este servicio es REST y se utiliza con el metodo POST, los datos enviados debe ser en formato JSON.
